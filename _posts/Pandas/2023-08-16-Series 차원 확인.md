# Series 차원 확인 :: Series.shape

<details>
    <summary>Reference</summary>
        Pandas In Action
</details>

```python
import pandas as pd

calories_info = {
    'Protein Bar' : 125,
    'Salade' : 215,
    'Chocolate Bar' : 406,
}

diet = pd.Series(calories_info)
diet
```

```
Protein Bar      125
Salade           215
Chocolate Bar    406
dtype: int64
```

```python
diet.shape # diet의 차원 
```

```
(3,)
```
