# Series 값의 유형 :: Series.dtype

<details>
    <summary>Reference</summary>
        Pandas In Action
</details>

```python
import pandas as pd

calories_info = {
    'Protein Bar' : 125,
    'Salade' : 215,
    'Chocolate Bar' : 406,
}

diet = pd.Series(calories_info)
diet
```

```
Protein Bar      125
Salade           215
Chocolate Bar    406
dtype: int64
```

```python
diet.dtype # diet의 값 데이터 유형
```

```
dtype('int64')
```
