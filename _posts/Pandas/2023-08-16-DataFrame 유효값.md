# df 행/열 별 유효값 개수 :: df.count()

```python
import pandas as pd

nba = pd.read_csv(
    './Data/nba.csv',
    parse_dates = ['Birthday']  # 'Birthday'의 데이터 유형을 날짜/시간(datetime)으로 강제 변환
)
nba
```

```python
df.count(
    axis = 0  # 0 or 'index' : 행 | 1 or 'columns' : 열
)
```

- 각 열 또는 행에 대해 NA가 아닌 셀 계산
- Option
  - axis : 축

```python
nba.count(axis = 0)
nba.count()          # nba의 각 열당 유효한 값의 개수
```
