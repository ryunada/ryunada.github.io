# df 유니크 값 확인 :: df.nunique( )

```python
df.nunique(
    axis = 0,        # 0 : 행 | 1 : 열
    dropna = True    # True : NaN 포함하지 않음 | False : NaN 포함
)
```

- 지정된 축의 개별 요소 수를 반환
- DataFrame은 Series와 달리 unique( )를 사용할 수 없음
- Option
  - axis : 사용할 축
  - dropna : 개수에 NaN 포함 여부

```python
nba.nunique() # nba 각 열의 유니크 값
```
